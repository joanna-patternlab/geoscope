# 🌍 GeoScope

GeoScope is a web application for exploring countries, geolocation data, and real-time time zones based on user input.

---

## 🚀 Live Demo

👉 https://geoscope-gray.vercel.app

---

## 🧠 About the Project

The app allows users to:
- check country details
- convert city → coordinates → real-time time
- reverse geocode coordinates → location data

---

## ✨ Features

### 🌍 Country lookup
- region
- capital
- population
- currency

### ⏰ Time by location
- city → coordinates → real-time time zone

### 📍 Reverse geocoding
- latitude & longitude → city, country, region

### 🗺 Map integration
- interactive map (Leaflet)

---

## 🛠 Tech Stack

### Frontend
- JavaScript (ES6+)
- jQuery
- SASS
- GSAP
- Leaflet

### Backend (Serverless)
- Vercel Functions
- API Ninjas (Geocoding)
- Time API (via proxy)
